Reporting a Missing Error

To help us diagnose and document a new error code accurately, please include the following details in your message if possible:

  • Exact Error Code / Stop Code: e.g., 0x0000007B, WHEA_UNCORRECTABLE_ERROR, DXGI_ERROR_DEVICE_REMOVED.
  • Operating System: e.g., Windows 11 Build 22631, Windows 10, etc.
  • System Hardware: Your processor, graphics card, RAM capacity, and motherboard model if relevant.
  • Context / Steps to Reproduce: What application was running, or what action triggered the crash?
  • Workarounds Tried: Any troubleshooting steps you have already attempted.
