Error Database

Hardware - BIOS
POST failure /error/POST-FAILURE/ Power-On Self Test failed. The PC will not pass the check and will not boot. CMOS Checksum error /error/CMOS-CHECKSUM-ERROR/ BIOS detected invalid data in CMOS. Settings have been lost. Violation of Secure Boot /error/SECURE-BOOT-VIOLATION/ Secure Boot blocked unsigned boot code from running. BIOS update failure /error/BIOS-UPDATE-FAIL/ Firmware update failed. The board does not need to boot. TPM chip not found /error/TPM-NOT-FOUND/ Windows 11 or BitLocker does not find the TPM 2.0 module. Wrong order of boot device in BIOS /error/BIOS-UEFI-BOOT-ORDER/ The computer is booting from the wrong device (USB instead of SSD or vice versa) due to the wrong sequence in the BIOS. Intel Management Engine (ME) corruption /error/ME-FIRMWARE-CORRUPTION/ The Intel ME firmware is corrupted, which leads to strange problems (PC won't turn off by itself, fans running at full speed, lack of sensors). Insufficient space in Option ROM BIOS /error/ROM-SPACE-EXHAUSTED/ The board has too many associated devices with custom ROMs (eg RAID controllers, multiple GPUs) and has run out of space to initialize them (POST). BIOS ROM checksum error /error/BIOS-ROM-ERR/ The BIOS ROM memory is corrupted or unreadable.
Hardware - CPU
WHEA uncorrectable error /error/WHEA-UNCORRECTABLE-ERROR/ Critical CPU or chipset hardware error (BSOD 0x124). Caused by voltage instability, overheating or overclocking. Clock watcher timeout /error/CLOCK-WATCHDOG-TIMEOUT/ The CPU core stopped responding to an interrupt request. Caused by overclocking or faulty driver. Temperature limitation of performance /error/CPU-THERMAL-THROTTLE/ The CPU reached the maximum safe temperature and activated throttling. Performance is reduced. MCE - Machine Check Exception /error/MACHINE-CHECK-EXCEPTION/ The CPU has detected a fatal hardware error. It could be a faulty processor or motherboard. Low CPU supply voltage /error/CPU-VCORE-LOW/ Powering the processor below the minimum causes instability and crashes. CPPC not supported /error/CPPC-NOT-SUPPORTED/ Collaborative Processor Performance Control is not active. Reduces the effectiveness of CPU boost. Limiting CPU performance with a performance limit of /error/POWER-LIMIT-THROTTLE/ The CPU is running below maximum power due to the PL1/PL2 limits of the motherboard or laptop. CPU Memory L1 Cache Error /error/L1-CACHE-ERROR/ The processor's internal L1 cache reported an unrecoverable data error. Extreme system instability. Activation of PROCHOT signal /error/PROCHOT-SIGNAL/ The Processor Hot (PROCHOT) signal has been activated, which causes extreme throttling, sometimes even at low temperatures. Thermal throttling of CPU performance /error/PROCHOT-1/ The processor slows down to avoid overheating and damage. Critical voltage drop across the core (Vdroop) /error/VCORE-DROP/ The power supply cascade does not supply sufficient voltage to the load. Exception while reading from L2/L3 cache /error/L2-CACHE-EXC/ The processor encountered corrupted data in its own cache. Processor microcode error /error/UCODE-0x1A/ The processor's internal instruction set encountered an unknown exception. Loss of Thread Synchronization (SMT) /error/SMT-SYNC-FAIL/ The logical cores have lost synchronization with each other. Integrated Memory Controller (IMC) failure /error/IMC-FAULT-00/ The controller inside the processor cannot communicate stably with the RAM. Crash when executing AVX instructions /error/AVX-PANIC/ Calculation error when using AVX/AVX2 wide vector instructions. Stuck on hibernation /error/C-STATE-HUNG/ The processor failed to properly wake up from sleep. Power cascade overheating (VRM) /error/VRM-TEMP-CRIT/ VRM sensors on the motherboard report critical temperatures limiting the CPU. Bad contact in processor socket /error/SOCKET-PIN-FLT/ The pins in the processor socket do not touch, which causes accidental disconnection of the lines. Exceeding the maximum thermal limit (TDP) /error/TDP-LIMIT-MAX/ The processor draws more power than the BIOS allows. Base clock instability (BCLK) /error/BCLK-JITTER/ The basic frequency generator on the board fluctuates. Branch prediction error /error/BRANCH-PRED-ER/ The jump prediction logic inside the CPU failed, typical of aging silicon. Coprocessor math error (FPU) /error/FPU-MATH-ERR/ The floating point unit returns imprecise results. Loss of internal bus synchronization /error/RING-BUS-FAIL/ The bus connecting the cores inside the processor collapsed. P-State transition failure /error/CPU-PSTATE-ERR/ Processor failed to transition to a requested performance state. AMD Infinity Fabric Instability (FCLK) /error/AMD-FCLK-INFINITY-ERR/ AMD systems crash or become unstable due to out of sync FCLK (Infinity Fabric clock) and MEMCLK.

Hardware - GPU
VIDEO TDR failure (0x116) /error/VIDEO-TDR-FAILURE/ GPU stopped responding, Windows reset it. Crash caused by driver or hardware failure. The screen goes black and the system restarts the driver. nvlddmkm.sys / TDR Crash /error/nvlddmkm.sys/ NVIDIA kernel driver crash. The screen goes black, the system tries to restart the driver. One of the most common NVIDIA BSOD codes. Code 43 - GPU Stopped Windows /error/CODE-43-GPU/ Windows stopped the GPU in Device Manager. The driver reported an unrecoverable error to the kernel. The most common GPU error after driver issues. DXGI - GPU stopped responding /error/DXGI-ERROR-DEVICE-HUNG/ DirectX has detected that the GPU has stopped processing commands. Game or app crashes. GPU fan failure /error/GPU-FAN-FAILURE/ The GPU fan stopped spinning. Causes rapid overheating and emergency shutdown. atikmpag.sys – AMD GPU crash /error/atikmpag.sys/ AMD GPU kernel driver crash. NVIDIA nvlddmkm.sys equivalent for AMD Radeon cards. Black screen with no signal /error/GPU-BLACK-SCREEN/ The monitor shows a black screen with no signal, although the PC appears to be running. VRAM corruption /error/VRAM-CORRUPTION/ The GPU video memory contains bad data. It causes artifacts, app crashes or BSODs. Graphical artifacts on screen /error/GPU-ARTIFACTING/ Random colored dots, stripes or triangles appear on the screen. Caused by VRAM error or overheating. PCIe slot bandwidth limitation /error/PCI-E-BANDWIDTH-ISSUE/ The graphics card runs on a lower generation or PCIe bus width (eg Gen3 x4 instead of Gen4 x16), which reduces performance. Video Memory Artifacts (Checkerboard) /error/VRAM-ART-01/ Colored squares or bars appear on the screen caused by a VRAM error. PCIe bus timer timeout /error/PCIE-TDR-116/ The graphics card did not respond to the motherboard within the timeout. Unstable graphics core clock /error/CORE-CLK-CRASH/ The core of the graphics card crashes when switching to 3D load. GPU power limit hit hard /error/PWR-LIM-THROT/ The card suddenly underclocks because it has hit the maximum consumption. Graphics fan controller failure /error/FAN-CRV-FAIL/ Graphics card fans do not spin or run at 100% regardless of temperature. Loss of video signal synchronization /error/DISP-SYNC-LOSS/ The image goes black for a second and pops up again, the connection with the monitor is lost. The graphics card has been physically disconnected from Windows /error/DEV-REMOVED/ The game will report that the graphics card has been disconnected from the system while running. Image flickering (G-Sync/FreeSync) /error/GSYNC-FLICKER/ The adaptive sync module causes the brightness to flicker violently. Shader compilation error /error/SHD-COMP-ERR/ The graphics card was unable to process the game's shader code. Desynchronization of linking multiple graphics /error/SLI-DESYNC/ Two graphics cards render images out of order (Micro-stuttering). GPU Hotspot Critical Temperature /error/HOTSPOT-110C/ The hottest sensor inside the core exceeded 110°C. View controller timeout and refresh /error/NVLDDMKM-TMOUT/ The graphics driver stopped responding for a second and Windows restarted. Extreme coil resonance on card /error/COIL-WHINE-MAX/ Coils make a deafening squeal due to high FPS or load. Corrupted graphics card firmware (VBIOS) /error/VBIOS-CORRUPT/ Graphics card won't boot due to damaged chip BIOS. Fan passive operation error /error/ZERO-RPM-FAIL/ The card does not turn off the fans even at zero load and low temperature. GPU fan controller failed /error/GPU-FAN-STOP/ The graphics card fan controller has failed, fans are not spinning. Monitor reports No Signal via HDMI /error/HDMI-NO-SIGNAL/ The screen is black and shows 'No Signal' even though the PC is on and the cable is connected. DisplayPort - signal loss or image flickering /error/DISPLAYPORT-NO-SYNC/ Monitor connected via DisplayPort randomly flickers, dims, or loses sync. Resizable BAR (SAM) causes performance degradation /error/RESIZABLE-BAR-ERROR/ FPS drops in some games after enabling Resizable BAR (AMD Smart Access Memory) or nVidia NVLink.

Thermal throttling of the VRM on the motherboard /error/VRM-OVERTEMP-THROT/ Processors with a high TDP (i9, Ryzen 9) reduce performance due to overheating of the power supply circuits (VRM) of the board. USB device handle request failed /error/USB-DEV-DESC-FAIL/ When connecting a USB device, the system failed to read its identifier and reports error code 43. Bluetooth adapter error 43 /error/BT-ERR-43/ Bluetooth has stopped working, the icon has disappeared and the device is reporting Code 43. Laptop battery is connected but not charging /error/BATT-NOT-CHARGING/ The adapter is on the network, the icon shows power, but the percentages do not increase. PCIe communication error /error/PCIE-ERROR/ Error on the PCIe line between the motherboard and the component. Motherboard VRM overheating /error/VRM-OVERHEAT/ The Voltage Regulator Module is overheating. Limits CPU performance or causes shutdown. PCIe bifurcation error /error/PCIE-BIFURCATION-ERROR/ The motherboard incorrectly distributes the PCIe lines for the adapter (NVME RAID, PLX switch). Corrupt CMOS checksum /error/CMOS-CHK-ERR/ The BIOS has lost its settings due to a dead coin-cell battery on the board. POST Critical Error (Code 55 - Memory) /error/POST-CODE-55/ The board failed to initialize RAM during boot. Failure to split PCIe lines (Bifurcation) /error/BIFURC-FAIL/ The splitter adapter (eg for multiple M.2 drives) did not get the correct PCIe lines. Missing certificates for Secure Boot /error/SEC-BOT-KEY-MS/ Safe boot platform keys deleted, can't boot into Windows 11. Security TPM chip verification failure /error/TPM-ATT-FAIL/ The Trusted Platform Module failed to verify the identity of the hardware. Unbalanced load of the board power phases /error/VRM-PHS-IMB/ One of the power supply phases on the board does not supply current, the others are critically overloaded. Critical failure of USB controller on board /error/USB-CTL-DEAD/ Some or all of the USB ports have completely stopped responding, the mouse and keyboard are not working. Released Passive Chipset Cooler (Chipset) /error/PCH-OVERHEAT/ The chipset on the board is overheating because the passive cooling has lost contact. Integrated sound card not found /error/AZA-AUD-MISS/ The integrated sound card (eg Realtek) completely disappeared from the device manager. Error uploading board firmware (BIOS) /error/BIOS-FLS-FAIL/ A corrupt installation file was detected while attempting to update the BIOS. Loss of connection with PCH chipset /error/DMI-LNK-DOWN/ Communication between the processor and the rest of the board collapses. Physical CMOS battery drain /error/CMOS-BAT-LEAK/ Physical damage to the board caused by button cell battery acid. Lighting Controller Short (RGB/ARGB) /error/ARGB-HDR-SHRT/ The motherboard has shorted on the 5V ARGB connector, the lighting is flashing or off. Sensor bus (LPC) freeze /error/LPC-BUS-HANG/ The sensor bus for temperature monitoring and fan control stopped responding. Burnt connector for connecting the fan /error/FAN-HDR-BURN/ Connecting a too powerful fan fried the PWM control circuit on the motherboard. USB device overcurrent status detected /error/USB-OVERCURRENT/ System shutting down after 15 seconds to protect motherboard from USB short circuit. PCIe generation - speed mismatch /error/PCIE-GEN-MISMATCH/ GPU or NVMe SSD running on PCIe 3.0 instead of PCIe 4.0/5.0, performance is significantly reduced.

Hardware - PSU
Source overvoltage /error/PSU-OVERVOLTAGE/ The power supply supplies a higher voltage than specified. Risk of component damage. Insufficient resource power /error/PSU-INSUFFICIENT/ The resource does not have time to cover the consumption of all components. Causes resets. Higher power ripple (Ripple) /error/PSU-RIPPLE-NOISE/ The source produces higher than permitted supply voltage ripple. It causes instability or damage to the capacitors on the board. Activation of overcurrent protection (OCP) /error/OCP-TRIGGERED/ The source detected a dangerously high current draw on some of the lines (typically 12V) and immediately shut down the computer just to be safe. Extreme coil whine (Coil Whine) /error/COIL-WHINE-EXTREME/ The source or graphics card emits a high-pitched, annoying buzzing sound, especially under load or at high FPS. Overvoltage protection (OVP) activated /error/OVP-TRIPPED/ The source immediately shut down because the voltage on some of the branches (12V, 5V, 3.3V) increased dangerously. Overcurrent protection (OCP) activated /error/OCP-TRIPPED/ The resource has shut down because one of the components (often the GPU) is drawing more current than the cable/branch can handle. Undervoltage protection (UVP) activated /error/UVP-TRIPPED/ The voltage fell below the critical limit, the system shut down as a precaution. Overload protection (OPP) activated /error/OPP-TRIPPED/ The total required system power exceeds the maximum power supply limit. Short circuit protection (SCP) activated /error/SCP-TRIPPED/ The source has detected a physical short circuit, immediately disconnects the power supply, the PC does not respond to the button at all. Overheat protection (OTP) activated /error/OTP-TRIPPED/ The source's internal temperature sensors exceeded the safe limit and the source shut down. Power Good (PG) signal failure /error/PG-SIGNAL-FAIL/ The power supply failed to confirm to the motherboard in time that the voltage is stable, therefore the board will not start (POST). Extreme Electrical Noise (Ripple) /error/RIPPLE-NOISE-MAX/ The DC power from the power supply contains too much AC interference (noise), causing instability and freezing of the PC. Soldering the power connector /error/PWR-CABLE-MELT/ The plastic connector (typically 12VHPWR or older PCIe) has melted due to contact resistance. Squealing coils from source /error/COIL-WHINE-PSU/ The power supply emits a very unpleasant high-frequency sound, especially when switched off or under load.

Overheating the power chip on DDR5 /error/DDR5-PMIC-TMP/ The Power Integrated Controller (PMIC) on the DDR5 memory has overheated. Memory calibration failure on PC startup /error/IMP-CAL-FAIL/ The motherboard is stuck in a loop while calibrating the memory signal impedance. MCE memory read error (Memory Read Error) /error/MCE-MEM-READ-ERR/ The processor detected an unrecoverable error while reading data from memory and wrote it to the MCE log.
